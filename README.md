# Orbit Drift

Минималистичная 2D-игра про корабль, космический дрейф и видимую гравитацию.

Игрок управляет не «машинкой в космосе», а траекторией: короткие импульсы двигателя, инерция, гравитационные поля, сканирование целей и перелёты между звёздными системами.

## Быстрый запуск

Открой `index.html` в браузере.

Для локального сервера:

```bash
npm start
```

Потом открой адрес, который покажет терминал.

## Проверка кода

```bash
npm run validate
```

Скрипт проверяет наличие ключевых файлов, порядок подключения JavaScript и синтаксис всех JS-файлов.

Дополнительная smoke-проверка запуска без браузера:

```bash
npm run smoke
```

## Сборка автономного HTML

```bash
npm run build:standalone
```

Сборка создаёт `dist/orbit-drift-standalone.html`. Это удобный вариант для быстрой отправки, но основной исходный код хранится в разнесённой архитектуре.

## Управление

### Компьютер

- удерживать мышь: тяга;
- отпустить: полёт по инерции;
- колесо мыши: приблизить/отдалить карту;
- `W` / `Space`: тяга;
- `A` / `D`: поворот;
- `S`: торможение;
- `R`: сменить цель;
- `G`: включить/выключить карту гравитации;
- `N`: включить/выключить навигацию;
- `M`: звук.

### Телефон

- удерживать экран: тяга;
- вести палец: направление;
- отпустить: инерционный дрейф;
- pinch двумя пальцами: масштаб;
- нижние кнопки: звук, навигация, поле, цель, масштаб.

## Концепт

- тёмный минималистичный космос;
- тонкие линии и геометрические объекты;
- корабль как маленький треугольник;
- гравитация видна и влияет на полёт;
- процедурные карты и разные звёздные системы;
- три уровня сложности;
- лёгкий запуск на телефоне и компьютере;
- без внешних библиотек и ассетов.
