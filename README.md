# Orbit Drift

Orbit Drift — небольшая 2D-игра про космический корабль, инерцию и полёт между звёздами. Вы выбираете курс, даёте короткие импульсы двигателем, используете притяжение небесных тел и стараетесь добраться до новой цели с запасом топлива.

## Быстрый запуск

### Вариант 1: открыть файл

Откройте `index.html` в браузере. Этого достаточно для игры.

### Вариант 2: локальный сервер

```bash
npm start
```

После запуска откройте:

```text
http://localhost:4173
```

Проект работает на обычных HTML, CSS и JavaScript, дополнительных библиотек для запуска нет.

## Цель игры

Летите от цели к цели, сканируйте планеты, звёзды, астероиды и кометы. За точные манёвры, удачные пролёты и выполнение заданий начисляются очки. Серия успешных целей повышает результат, а ошибки расходуют топливо, перегревают корабль или создают опасную нагрузку.

На экране видны карта притяжения, прогноз траектории, запас топлива, нагрев, нагрузка и текущая цель. Притяжение помогает разгоняться и менять курс, если входить в поле под подходящим углом.

## Управление

### Компьютер

- Удерживать левую кнопку мыши: тяга в сторону указателя.
- Отпустить мышь: полёт по инерции.
- Колесо мыши: приблизить или отдалить карту.
- `W`, `Space`, `ArrowUp`: тяга.
- `A`, `D`, `ArrowLeft`, `ArrowRight`: поворот.
- `S`, `ArrowDown`: торможение.
- `R`: сменить цель.
- `G`: включить или выключить карту притяжения.
- `N`: включить или выключить навигацию.
- `M`: включить или выключить звук.

### Телефон

- Удерживать экран: тяга.
- Вести палец по экрану: направление тяги.
- Отпустить экран: полёт по инерции.
- Развести или свести два пальца: масштаб карты.
- Нижние кнопки: звук, навигация, карта притяжения, смена цели и масштаб.

## Сложность

В меню доступны три режима:

- `Легко`: ближние цели, больше топлива и мягкая цена ошибок.
- `Нормально`: сбалансированный режим.
- `Сложно`: дальние цели, слабее тяга, выше риск перегрева и нагрузки.

## Проверка проекта

Проверить структуру файлов, порядок подключения JavaScript и синтаксис:

```bash
npm run validate
```

Запустить smoke-проверку игровой логики без браузера:

```bash
npm run smoke
```

## Сборка одного HTML-файла

Собрать один HTML-файл для передачи или публикации:

```bash
npm run build:standalone
```

Готовый файл появится здесь:

```text
dist/orbit-drift-standalone.html
```
