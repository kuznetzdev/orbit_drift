# JavaScript modules

Файлы подключаются обычными `<script defer>` в порядке нумерации.

Причина: игра должна открываться напрямую из ZIP или локальной папки без сервера и сборщика. Это важнее, чем строгие ES-module импорты на текущем этапе.

Не меняй порядок файлов без проверки `npm run validate` и ручного запуска игры.
